======================================================
  기준 음원 파일 배치 안내
======================================================

이 폴더에 아래 파일을 배치하면 cross-correlation(교차 상관)
방식으로 인식음과 중간음을 정확하게 감지합니다.

------------------------------------------------------
파일 이름 (확장자 무관)
------------------------------------------------------

  recognition_sound.wav   — 음성 인식음
                            (사용자 발화 후 기기가 인식했을 때 나는 짧은 소리)

  middle_sound.wav        — 중간음
                            (응답 생성 중 나오는 처리 중 알림음)

※ .wav 외에도 .m4a / .mp3 / .flac / .aac / .ogg 확장자 모두 사용 가능합니다.
※ 파일명(확장자 제외)이 정확히 위와 일치해야 합니다.

------------------------------------------------------
기준 파일이 없을 경우
------------------------------------------------------

파일이 없으면 RMS 에너지 기반 자동 감지(폴백 모드)로 동작합니다.
감지 정확도가 낮을 수 있으므로, 가능하면 기준 음원을 준비하세요.

------------------------------------------------------
감도 조정
------------------------------------------------------

config.py 의 REFERENCE_THRESHOLD 값을 조정하세요.
  - 기본값: 0.55
  - 낮출수록: 더 많이 감지 (오감지 위험)
  - 높일수록: 더 엄격하게 감지 (미감지 위험)
